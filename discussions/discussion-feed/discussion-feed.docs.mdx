A discussion feed UI component for querying and displaying discussions in the Acme platform.

## Get started

Install and use the component in your code:

```js
import { DiscussionFeed } from '@raptants/discussions.discussion-feed';

<DiscussionFeed />
```

## Testing

During testing the compoennt will display mock data. To get mock data to be rendered make sure to wrap the component with the apollo `MockedProvider`.
