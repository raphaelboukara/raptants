A React Hook for fetching discussions data on the Acme platform.

## Get started

Install and use the hook in your component:

```tsx
export function MyComponent() {
  const response = useDiscussionList();
  return (<> found {announcements.length} discussions</>);
}
```

## Testing

Upon testing, the component will return a set of mock announcement data.
