The Acme composition. Composes the Acme micro-services and micro-frontends into a unified platform.

## Run the server 

Import the platform and use the it in your workspace:

```bash
bit use my-project-platform
```

Run the server:
```bash
bit run my-project-platform
```

## Compose new apps

You can introduce new micro-services or micro-frontends to the platform by composing the in the `my-project-platform.bit-app.ts`:

```ts
import { Platform } from '@bitdev/platforms.platform';

export const MyPlatform = Platform.from({
  name: 'my-project-platform',

  backends: {
    main: new AcmeGateway(),
    services: [
      DiscussionService,
    ]
  },
});
```
